Spring boot WebFlux!

Spring WebFlux est un framework Web entièrement non bloquant et basé sur des annotations, construit sur Project Reactor, qui permet de créer des applications réactives sur la couche HTTP. WebFlux utilise une nouvelle fonctionnalité de fonctions de routeur pour appliquer une programmation fonctionnelle à la couche Web et contourner les contrôleurs déclaratifs et les RequestMappings. WebFlux vous oblige à importer Reactor en tant que dépendance principale.
